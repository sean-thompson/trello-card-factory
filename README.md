# Card Factory for Trello

Card Factory is a Trello Power-Up that lets you turn any card into a template for quickly creating new cards. Drop an image onto a factory card, give it a name, and a new card is created with all the right settings already applied.

## What it does

Set up a card the way you want new cards to look — add labels, assign members, write a description, create checklists, set a due date. Then enable Card Factory on that card. From that point on, any time you drop an image onto it, you'll be prompted to create a new card from that image. The new card gets all the attributes you chose copied over automatically.

This is especially useful when you regularly create cards that share the same labels, members, or checklists but each need a unique image — think product listings, asset reviews, content pipelines, or anything where the setup is repetitive but the content is unique.

## How to use it

### Setting up a factory card

1. Open any card on your board
2. Click the **Card Factory** button in the Power-Up section
3. Click **Enable Card Factory**
4. Choose which attributes to copy to new cards (labels, members, description, due date, checklists, custom fields)
5. Pick which list new cards should be created in — either the same list as the factory card, or any other list on the board
6. Click **Save**

The card will now show a **Factory** badge so you can tell at a glance which cards are factories.

### Creating cards from images

1. Drag and drop an image onto your factory card (or attach one the usual way)
2. A small form will appear on the card showing the image filename as a suggested card name
3. Edit the name if you'd like, then click the **checkmark** to create the card
4. The new card will appear in your chosen list with all the configured attributes and the image attached

If you don't want to create a card from a particular image, click the **X** to dismiss it — the image will stay on the factory card like a normal attachment.

### Changing settings

Click the **Card Factory** button again at any time to change which attributes are copied or where new cards are created. You can also disable the factory entirely from the same menu.
